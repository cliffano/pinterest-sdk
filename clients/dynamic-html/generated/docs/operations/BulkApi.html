This is the API

bulkDownload/create

Create an asynchronous report that may include information on campaigns, ad groups, product groups, ads, and/or keywords; can filter by campaigns. Though the entities may be active, archived, or paused, only active entities will return data.

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/bulk/download

HTTP Method

POST

Response Type

BulkDownloadResponse

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Body: BulkDownloadRequest BulkDownloadRequest(BulkDownloadRequest)

    Parameters to get ad entities in bulk

bulkRequest/get

Get the status of a bulk request by <code>request_id</code>, along with a download URL that will allow you to download the new or updated entity data (campaigns, ad groups, product groups, ads, or keywords).

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/bulk/{bulk_request_id}

HTTP Method

GET

Response Type

BulkUpsertStatusResponse

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Path: bulk_request_id String

    Unique identifier of a bulk upsert request.

  • Query: include_details Boolean

    if set to True then attach the errors/details to all the requests

bulkUpsert/create

Either create or update any combination of campaigns, ad groups, product groups, ads, or keywords. Note that this request will be processed asynchronously; the response will include a <code>request_id</code> that can be used to obtain the status of the request.

URL

https://api.pinterest.com/v5/ad_accounts/{ad_account_id}/bulk/upsert

HTTP Method

POST

Response Type

BulkUpsertResponse

Parameters

  • Path: ad_account_id String

    Unique identifier of an ad account.

  • Body: BulkUpsertRequest BulkUpsertRequest(BulkUpsertRequest)

    Parameters to get create/update ad entities in bulk
